# 🌙 Moondesk Design System

Sistema de diseño oficial de **Moondesk** — una plataforma SaaS profesional diseñada para claridad, foco y eficiencia.

---

## 🎨 Visión General

Moondesk Design System define los fundamentos visuales, tokens de diseño y guías de componentes para construir interfaces consistentes y de alta calidad en todo el ecosistema Moondesk.

### Características principales

- **Arquitectura de tokens semánticos** — Sistema de dos capas (primitivo → semántico) que separa valores crudos de sus roles de uso
- **Soporte Light & Dark Mode** — Tematización completa vía tokens semánticos
- **Basado en Material Design 3** — Roles de color extendidos con identidad de marca propia
- **Tipografía Inter** — Fuente optimizada para interfaces, con escala tipográfica consistente
